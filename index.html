Coding Projects

ReadROOT Python Package

This project was made for the physics department of McGill university over the summer 2023. This Python package was sorely made to complement the CoMPASS software made by CAEN. This package contains a GUI which allows you to reload the CoMPASS saved files and graph again all of the histograms available in the CoMPASS software.

More about the package and its different functionalities can be learned in the Wiki of the Github repository.
Repository

Honkai Impact 3rd Database Bot

This project has been ongoing since October 2022. This project includes html parsing of the Stigmata Wiki for Honkai Impact 3rd. The result of the html parsing makes up the database of the bot, saved in memory, meaning that the database can be reparsed at anytime if updates are to be made. This bot has since August 2023 been online without many issues and its status can be tracked over on Uptime Robot.
